WELCOME TO THE CODING GHOST TREE

Data compression, source coding or bit-rate reduction involves encoding information using fewer bits than the original representation. Compression can be either lossy or lossless. Lossless compression reduces bits by identifying and eliminating statistical redundancy. No information is lost in lossless compression. Lossy compression reduces bits by removing unnecessary or less important information. There are several data compression technique we have come to know so far such as Huffman coding , bit coding , byte coding. We want to design an interface through which user can use any compression technique to code or decode a value.
